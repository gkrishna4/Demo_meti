Configure online EPEL repository in RHEL 8:
dnf install https://dl.fedoraproject.org/pub/epel/epel-release-latest-8.noarch.rpm
rpm -qa | grep epel
dnf --disablerepo="*" --enablerepo="epel" list available
dnf --disablerepo="*" --enablerepo="epel" list available | grep htop
dnf install htop	(for testing)

Configure online EPEL repository in CentOS 8:
yum install epel-release
yum repolist

Disable warning message of Red Hat Subscription Management:
vi /etc/yum/pluginconf.d/subscription-manager.conf
replace "enabled=1" with "enabled=0"
